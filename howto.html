Custom Emotes Integration Guide

  1. Add the following line to your HTML page within the <head> section: 
    <meta http-equiv="Content-Security-Policy" content="upgrade-insecure-requests">
<script src="https://ion606.github.io/streamelements/injectEmotes.js"></script>
  2. Use the following JavaScript code snippet where you need the emotes to be placed: 
    const msgInner = await placeCustomEmotes(messageContent);
  3. Ensure the enclosing function is asynchronous. If it's not, modify it by adding async before the function definition. For example: 
    function addMessage(username, badges, message, isAction, uid, msgId) {
    will become 
    async function addMessage(username, badges, message, isAction, uid, msgId) {
